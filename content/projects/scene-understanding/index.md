## Overview

This project develops machine learning algorithms for 3D object recognition using point cloud data captured from ARKit and ARCore frameworks, enabling intelligent scene understanding and collaborative augmented reality experiences on mobile devices. In partnership with AWE Company, the work targets real-world AR applications that can semantically understand their environment in real time.
